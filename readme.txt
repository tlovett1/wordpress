Adds the ability to create Knotch widgets from posts.

== Installation ==
1. Copy everything to a \"knotch\" directory in the /wp-content/plugins/ directory.
2. Go to the new Knotch settings menu in the Wordpress dashboard and enter your credentials.
3. You should now be able to set up Knotch widgets while editing posts by enabling the menu under \"Screen Options\".
